# 🌐 SZABIST Campus Network Infrastructure

A scalable and secure **enterprise campus network** designed in **Cisco Packet Tracer**. The project implements a three-tier hierarchical architecture connecting multiple campus buildings with centralized routing, switching, IP addressing, and security services.

---

## ✨ Features

- 🏢 Three-Tier Hierarchical Network Design
- 🌍 Multi-Building Campus Connectivity
- 🔀 VLAN Segmentation & IEEE 802.1Q Trunking
- 🌐 VLSM-Based IP Addressing
- 🔁 Inter-VLAN Routing (Router-on-a-Stick)
- 📡 OSPF Dynamic Routing
- 📥 Centralized DHCP Services
- 🌐 NAT/PAT for Internet Connectivity
- 🔒 Access Control Lists (ACLs)
- 🛡️ Port Security with Sticky MAC
- 💻 Remote Device Management (Telnet)
- 🖥️ Enterprise Data Center Integration

## 🏗️ Network Architecture

The network follows a **Three-Tier Hierarchical Model**, consisting of:

- **Core Layer** – Centralized backbone for campus communication
- **Distribution Layer** – Aggregates traffic from multiple buildings
- **Access Layer** – Connects end-user devices and departmental networks

The infrastructure integrates multiple campus buildings through a centralized core while providing secure communication, dynamic routing, automated IP allocation, and policy-based network security.
